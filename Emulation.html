Emulation

I've always been interested in low-level programming and CPU architecture so I've begun a project focused on emulation to further my knowledge in these topics. The primary goal of the project is to make three emulators. I've chosen the Intel 8080 CPU, the Zilog Z80 CPU (used for the ZX Spectrum), and the Game Boy as these are individual projects which start fairly simple yet can build upon each other. For each emulator, I intend to make at least one small game written in the hardware's assembly language. The goal of the project is to spend more time writing and reading low-level code, particularly in C and various assembly languages. I'm not only doing this because I'm interested in these topics, I believe that a better understanding of exactly what's going on when code is being run makes for a much improved programmer in any language. I will update this page as each step of the project is completed.

The first step of this project is creating an Intel 8080 emulator which will be able to run games like Space Invaders and Gun Fight.

Disassembler

To begin the Intel 8080 project, I have made a disassembler for ROM files. This is useful as it not only allows me read the source code from each ROM but also provides an opportunity to read through all of the assembly commands which helps undestand the CPU architecture. This understanding will be vital in the next part of the project, creating the emulator.
