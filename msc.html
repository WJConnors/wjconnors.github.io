Computer Game Engineering MSc

OpenGL Graphics Project

This project is built to demonstrate advanced graphics techniques created with OpenGL and shaders. It contains multiple scenes which can be switched between that each demonstrate different effects. These are some of the effects and technical details created for the project:

  • Generated landscapes with multiple blended textures
  • Standard lighting techniques using ambient, diffuse, and specular components
  • Skybox textures used in combination with environment mapping to show water reflections
  • Loading and animating meshes with the appropriate textures
  • Deferred rendering and advanced buffers to allow for:
    • A post-processing blur effect
    • Dynamically rendering up to 256 moving light sources
    • A split-screen effect to show both scenes simultaneously

Game of Life C++

This university project demonstrates the use of advanced C++ features to create an optimized simulation for Conway's Game of Life. It can accurately run simulations on varying board sizes and search for specific patterns, including spaceships. This codebase has been built from scratch using object-orientation and utilizes many C++ features including operator overloading, templates, threading, and interfaces.
