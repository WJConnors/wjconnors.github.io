Shot in the Dark

View Github View on itch.io

Shot in the Dark is a three-person Unreal Engine project made in a week for the Newcastle University Game Development Society Winter Jam 2025. It is a short survival horror game where you explore a mine with a torch that has a limited battery. There is a monster hunting you as you explore and the goal of the game is to collect weapon parts to eventually defeat the monster while ensuring your torch battery does not run out. The goal of this project was to quickly gain familiarity with Unreal Engine as well as gain experience working in a team.

My Contributions

I made the following contributions to the project:

  • Creating the player character, their movement, and their interaction with world objects
  • Building and programming the UI
  • Implementing the game's framework surrounding the gameplay
  • Designing and implementing the game's objectives as well as the win and lose conditions
  • Adding the game's sound effects
Due to the time constraints and the goal of gaining familiarity with the engine, this project was built in blueprints. It was interesting using such a different style of programming with unique benefits and drawbacks. I enjoyed how the bespoke language allowed for quick prototyping of and iteration upon game mechanics. I also learned very quickly that it requires very careful use to prevent code becoming an unreadable mess of connections.
