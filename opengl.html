OpenGL Graphics Project

View Github

This project is built to demonstrate advanced graphics techniques created with OpenGL and shaders. It contains multiple scenes which can be switched between that each demonstrate different effects.

Creating this scene involved rendering with a series of frame buffers to draw the scene any any effects. An example of shader code and part of the render pipeline is below.

This scene uses deferred rendering to create 256 lights where the position and colour are randomly generate which light up the landscape and the 1024 spheres acting as snow.

The scene contains water which has implementations of partial transparency and reflections.

The scene used shaders to create a transition effect when the main scene flythrough is switching between scenes to mask the transition.
