Shot in the Dark

Shot in the Dark is a three person Unreal Engine project made in a week for the Newcastle University Game Development Society Winter Jam 2025. It is a short survival horror game where you explore a mine with a torch that has a limited battery. There is a monster hunting you as you explore and the goal of the game is to collect weapon parts to eventually defeat the monster while ensuring your torch battery does not run out.
I made the following contributions to the project:

  • Creating the player character and their movement
  • Creating the torch and the battery pickups
  • Building and programming the UI
  • Implementing the game's framework surrounding the gameplay
  • Designing and implementing the game's objectives as well as the win and lose conditions
  • Adding the game's sound effects

OpenGL Graphics Project

This project is a project built to demonstrate advanced graphics techniques created with OpenGL and shaders. It contains multiple scenes which can be switched between that each demonstrate different effects. These are some of the effects and technical details created for the project:

  • Generated landscaped with multiple blended textures
  • Standard lighting techniques making use of ambient, diffuse, and specular components
  • Skybox textures used in combination with environment mapping to show water reflections
  • Loading in and animating meshes with the appropriate textures
  • Deferred rendering and multiple advanced buffers have been implemented to allow for the following features
    • A post processing blur effect
    • Dynamically rendering up to 256 moving light sources
    • Showing both scenes with a split screen effect

Game of Life C++

This is a university project which demonstrates use of advanced C++ features in order to create an optimised simulation for Conway's Game of Life. It is able to accurately run simulations on varying board sizes and search for specific patterns including space ships. This is a codebase which has been built from scratch using object-orientation and makes use of many C++ features including operator overloading, templates, threading, and interfaces.
